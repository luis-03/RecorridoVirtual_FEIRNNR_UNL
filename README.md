# RecorridoVirtual_FEIRNNR_UNL
Recorrido virtual inmersivo y fotorrealista de los exteriores de la FEIRNNR – UNL, desarrollado mediante fotogrametría híbrida y tecnologías de realidad virtual. El proyecto integra modelos 3D optimizados en un motor gráfico, permitiendo navegación libre en primera persona y evaluando la experiencia del usuario a través de la sensación de presencia
